# The PODM project
A study of the association between occupational particle exposure and risk of type 2 diabetes.
## Data sources
This project relies on data from the Swedish Construction Workers' Cohort and several national registries. The data is not shared because of its sensitive nature. The base cohort data are held at Umeå University (https://www.umu.se/forskning/infrastruktur/bygghalsokohorten/). The registry data are held by the relevant Swedish authorities and are available to researchers subject to appropriate ethical permission.
## Code
All code in this project is written in R and free to re-use.
