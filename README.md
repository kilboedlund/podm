# The PODM project
This is a study of the association between occupational particle exposure and risk of type 2 diabetes.
## Data sources
This project relies on data from the Swedish Construction Workers' Cohort and several national registries. The data are not shared because of their sensitive nature. The baseline cohort data are held at Umeå University (<link>https://www.umu.se/forskning/infrastruktur/bygghalsokohorten/</link>). The follow-up registry data are held by the relevant Swedish authorities (Statistics Sweden, National Board of Health and Welfare). Additional data were obtained from the Swedish National Diabetes Register. Data are available from the respective data holders, for researchers with an agreement with the cohort holders and subject to appropriate ethical permissions valid in Sweden.
## Code
All code in this project is written in R and free to re-use.
